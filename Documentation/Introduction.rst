What does it do?
================

The **maispace/mai-base** extension is the base extension for Maispace TYPO3 projects.
It provides developer utilities, TCA registration helpers, and backend enhancements
for TYPO3 CMS 13.

Features
========

- **Backend Enhancements** – Adds "Save and Close" and record navigation buttons to editing forms.
- **TCA Registration Helpers** – Fluent builder classes for CType, Doktype, Field, Table, and Palette registration.
- **Utility Classes** – QueryDebugger and ActiveExtensionConfigurationLoader for common development tasks.
- **Backend Layouts** – Preconfigured backend layouts with multiple content columns.
- **TypoScript** – Plugin configuration, PID management, and content rendering libraries.
